# Customer Churn Analysis (Power BI)

## 📊 Overview
This project analyzes customer churn behavior using Power BI, focusing on identifying when users are most likely to churn.

## 🔍 Key Insights
- Overall churn rate is ~26%
- Churn is highest in early lifecycle users (0–3 months: 56%)
- Churn decreases significantly as tenure increases (12+ months: 17%)
- Early engagement is critical for retention

## 🛠 Tools Used
- Power BI
- DAX
- Data Visualization
